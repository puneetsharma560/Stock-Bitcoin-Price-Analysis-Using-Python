# 📊 Stock & Bitcoin Price Analysis Using Python

## Project Overview

This project analyzes historical Stock and Bitcoin price data using Python. It performs Exploratory Data Analysis (EDA), calculates technical indicators, evaluates return and volatility, and applies statistical tests to understand market behavior.

---

## Objectives

- Download historical Stock and Bitcoin price data
- Perform Exploratory Data Analysis (EDA)
- Calculate Daily Returns
- Analyze Cumulative Returns
- Measure Volatility
- Generate Technical Indicators
- Perform Statistical Tests
- Visualize market trends and distributions

---

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- SciPy
- yFinance

---

## Key Features

- Historical data collection
- Data cleaning and preprocessing
- Daily Return Analysis
- Cumulative Return Analysis
- Volatility Analysis
- Simple Moving Average (SMA)
- Exponential Moving Average (EMA)
- Relative Strength Index (RSI)
- Correlation Analysis
- Return Distribution Analysis
- Augmented Dickey-Fuller (ADF) Test
- Shapiro-Wilk Normality Test
- Jarque-Bera Normality Test
